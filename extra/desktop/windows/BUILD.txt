Install enviroment:
1. Download Strawberry Perl http://strawberryperl.com/
2. Download PAR http://par.perl.org/wiki/Main_Page
3. Execute "cpan PAR::Packer"
4. Download http://ftp.gnome.org/pub/gnome/binaries/win32/dependencies/pkg-config_0.23-3_win32.zip
5. Extract and copy pkg-config.exe to some directory in PATH
6. Execute "cpan Net:LDAP"
